TRIMMING
Trimming is a Maven plugin
Run command
Java bytecode debloating analysis
Analysis result
Bloated bytecode analysis report
Step1: Downloads
        Tool link:     Trimming
Step2: Install
        Windows:     Install.bat
        Linux/Mac:     Install
Step3: Run
        mvn neu.lab:trimming:1.0:debloat
Optional Parameters
Name Type Description
entryPointStrategy String Trimming offers four entry method strategies for debloating: Main, ProjectCoreMethod, Test, and ProjectCoreMethodAndTest. Default value is: ProjectCoreMethodAndTest
